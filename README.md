# 全国与城市二手房交易知识库网页

全国通则及北京、深圳、广州二手房交易政策的单页检索站点，展示：

- 全国统一的产权登记、限制转让、共有/抵押/租赁和定金规则
- 全国住房信贷底线、公积金利率、登记费、经纪收费和资金监管原则
- 购房资格和地域差异
- 房屋可交易性核验
- 商贷、公积金和政策最低首付
- 契税、增值税和个人所得税
- 买方现金需求计算公式与示例
- 政策版本演变、来源新鲜度和官方依据

网页只负责浏览与检索已审核知识，不把自身定位为交互式交易计算器；AI/Agent 应读取 Wiki 规则、版本和来源后完成场景判断。

## 数据来源

事实源为工作区上级目录的 `wiki/`。当前网页发布版本为 `2026.07.15-r2`，对应：

- 全国：`cn@2026-07-15.1`
- 北京：`bj@2025-12-24.1`
- 深圳：`sz@2026-04-30.1`
- 广州：`gz@2024-09-30.1`

页面结构化数据位于 `app/knowledge-data.ts`。政策更新完成来源监测、影响分析、规则修改、黄金用例和人工审核后，应同步更新此文件并重新构建发布；网页不直接把未经审核的官网变化自动转为交易结论。


访问 `http://localhost:3000/`。
